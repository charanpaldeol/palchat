# Hello

A minimal Astro site with a single Hello page.

## Run locally

```bash
cd myastosite
npm install
npm run dev
```

Open http://localhost:4321

## Build

```bash
cd myastosite
npm run build
```

## Deploy

The site is configured for Vercel (`@astrojs/vercel`). Deploy from the `myastosite` directory or connect the repo to Vercel with root set to `myastosite`.
