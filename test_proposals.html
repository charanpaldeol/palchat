Proposals Page Testing

Test Instructions

  1. Open the browser console (F12)
  2. Navigate to the proposals page: http://localhost:4322/proposals
  3. Copy and paste the test script from test_proposals_page.js
  4. Run proposalTests.runAllTests()
  5. Check the console output for test results

Quick Test Commands

Run all tests: proposalTests.runAllTests()

Test Details button: proposalTests.manualTestDetails()

Test Accept button: proposalTests.manualTestAccept()

Test Reject button: proposalTests.manualTestReject()

Expected Behavior

  • Details Button: Opens modal with proposal details
  • Accept Button: Shows confirmation, then changes status to "executed"
  • Reject Button: Prompts for reason, then changes status to "rejected"
  • Test Button: Shows notification with instructions
  • Filters: Filter proposals by status and type

Troubleshooting

Buttons not working?

Check if the functions are defined in the console: typeof viewProposal

Modal not showing?

Check if modal element exists: document.getElementById('proposal-modal')

No proposals showing?

Check if proposals data is loaded: console.log(proposals)

Console errors?

Look for red error messages in the console and check the network tab for failed requests
